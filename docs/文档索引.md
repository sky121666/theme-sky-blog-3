## 统一规则

1. `README.md` 不再承载长篇功能方案。
2. 除 `README.md` 外，文档文件名统一使用中文。
3. 文档内项目文件链接统一使用**项目根相对地址**，例如：
   - `[/settings.yaml](/settings.yaml)`
   - `[/src/apps/photos/entry.js](/src/apps/photos/entry.js)`
4. 功能变化优先更新对应专题文档，不要只改 README。
