# 豆瓣

## 结论

`/douban` 是主题自定义的 macOS App 壳层页面，用于展示 `plugin-douban` 的个人书影音归档。

当前契约确认到 `plugin-douban` `1.2.5`。主题消费插件公开 API 渲染页面，不复用插件默认 `modules/douban` UI 片段。

## 数据契约

主题侧使用的 API 基准：

| 接口 | 用途 |
| --- | --- |
| `/apis/api.douban.moony.la/v1alpha1/doubanmovies` | 列表、分页、搜索和状态统计 |
| `/apis/api.douban.moony.la/v1alpha1/doubanmovies/-/types` | 顶部类型导航 |
| `/apis/api.douban.moony.la/v1alpha1/doubanmovies/-/genres` | 题材筛选 |

列表查询参数：

| 参数 | 来源 |
| --- | --- |
| `page` | 当前页码 |
| `size` | 固定分页大小，当前为 `20` |
| `type` | 当前类型，默认 `movie` |
| `status` | 收藏状态，非全部时传递 `mark / doing / done` |
| `genre` | 当前题材，非全部时传递 |
| `keyword` | 桌面端和移动端搜索框同步输入 |

主题消费 `DoubanMovie.spec` 和 `faves` 字段，核心展示包括名称、年份、海报、豆瓣评分、类型题材、收藏状态、个人评星、记录时间、短评和详情链接。

## 页面能力

- 类型切换：电影、图书、音乐、游戏、舞台剧，优先使用 `types` 接口返回结果，接口失败时使用前端兜底类型。
- 状态筛选：按当前类型切换“想看/想读/想听/想玩”“在看/在读/在听/在玩”“看过/读过/听过/玩过”等文案。
- 题材筛选：按 `genres` 接口返回的题材动态渲染。
- 搜索：桌面端和移动端输入框联动，输入后延迟刷新。
- 展示模式：支持海报网格和列表视图。
- 分页：按“载入更多项目”追加下一页。
- Quick Look：点击条目或空格键打开预览，展示海报、评分、题材、个人评星和短评。
- 键盘导航：方向键在当前可见条目之间移动焦点。
- 空状态和错误状态：空筛选结果展示空状态；接口失败提示检查插件安装、同步和匿名 API 权限。

## 后台配置

豆瓣页有独立后台配置组：

| 配置 | 用途 | 默认行为 |
| --- | --- | --- |
| `douban.profile.icon` | 页面头像图标 | 留空使用默认书影音图标 |
| `douban.profile.display_name` | 页面标题 | 留空使用当前页面标题 |
| `douban.profile.subtitle` | 页面说明 | 留空使用默认说明 |
| `douban.style.color_mode` | 配色模式 | 默认 `douban`，可切到 `theme` 跟随主题强调色 |

图标配置接受 Iconify 输出的 SVG 值。运行时会提取合法 SVG；如果传入的是损坏的 JSON 或非 SVG 字符串，则保留默认图标。

## 移动端

移动端按独立 App 收口：

- 隐藏全局 header 和 Dock。
- 窗口铺满视口。
- 使用移动端标题栏、类型横向导航和轻量搜索框。
- 保留网格/列表切换，但不显示桌面端交通灯工具栏。

## 边界

- 主题只负责前台展示，不提供豆瓣同步、写入、编辑或删除能力。
- 封面稳定性主要由 `plugin-douban` 同步结果和封面代理配置决定；主题端对图片使用 `referrerpolicy="no-referrer"` 并提供无图兜底。
- `/douban` 的数据真实性以插件公开 API 为准，主题不在前端伪造条目。
