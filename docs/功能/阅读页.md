# 阅读页

## 结论

阅读页是独立 App，承接：

- 文章页
- 独立单页

当前目录名是：

- [/src/apps/reader](/src/apps/reader)

协议名是：

- `pageApp = reader`
- `pageMode = browser-reader`
- `windowVariant = browser`

## 功能边界

阅读页 App 负责：

- 文章正文壳
- 目录
- 点赞
- 评论区容器
- 上下篇导航

不负责：

- 顶栏
- Dock
- 主窗口壳层
- PJAX 总线

这些仍由 `desktop-shell` 负责。

## 什么时候改这里

这些需求应该优先改阅读页 feature：

1. 正文排版
2. 目录行为
3. 点赞逻辑
4. 评论区容器样式
5. 上下篇卡片

## 我想配什么，先配哪里

阅读页本身没有单独主题设置组。
如果你只是想影响阅读页外围行为，优先检查：

1. 评论组件颜色变量：看 shell token
2. 分享与标题栏行为：看 [/docs/功能/桌面壳层.md](/docs/功能/桌面壳层.md)
